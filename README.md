# Azure Infrastructure Deployment with ARM/Bicep

## Project Overview
This project demonstrates the design and deployment of a secure, scalable Azure infrastructure using Infrastructure as Code (IaC) with ARM/Bicep.

## Key Features
- Virtual Network with multiple subnets
- Network Security Groups (NSGs)
- Windows & Linux Virtual Machines
- Azure Key Vault for secrets management
- Storage Account
- Recovery Services Vault (Azure Backup)
- Fully automated deployment using Bicep
